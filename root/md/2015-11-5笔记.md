##arguments
``` javascipt
	function a(){
		consloe.log(Arguments);
	}
	a(1,2,3);
```
如下为匿名函数

``` javascipt
	(function(){
		consloe.log(Arguments);
	})(1,2,3);
```
输出都为1，2，3；
